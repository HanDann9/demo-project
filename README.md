# Node.js & ExpressJs

### Version: 1.0.0

### Usage

```sh
$ npm install
```

### 1. Create database in mysql

### 2. Running Migrations

```sh
$ npx sequelize-cli db:migrate
```

### 3. Running Seeds

```sh
$ npx sequelize-cli db:seed:all
```

### 4. Start project

```sh
$ npm start
# Visit http://localhost:8888
```
